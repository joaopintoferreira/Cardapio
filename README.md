# 🍽️ Cardápio Digital - Sistema de Pedidos

[![Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black)](https://vercel.com)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

> Um sistema interativo de cardápio digital onde clientes podem visualizar os pratos e realizar pedidos diretamente pelo site.

## 📸 Demonstração

🔗 **URL do projeto:** [cardapio-two-area.vercel.app](https://cardapio-two-area.vercel.app)

> ⚠️ *Se o link não estiver funcionando, o deploy pode ter expirado — siga as instruções de recuperação acima.*

## ✨ Funcionalidades

- ✅ Cardápio digital completo com categorias
- ✅ Sistema de pedidos integrado
- ✅ Interface responsiva (funciona em celular/tablet/desktop)
- ✅ Visualização de itens com preços e descrições
- ✅ Carrinho de pedidos

## 🛠️ Tecnologias utilizadas

| Tecnologia | Finalidade |
|------------|------------|
| HTML5 / CSS3 | Estrutura e estilização |
| JavaScript | Interatividade e lógica de pedidos |
| Vercel | Hospedagem e deploy contínuo |
| Git/GitHub | Versionamento de código |
